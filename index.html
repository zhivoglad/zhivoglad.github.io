CSS свойство - background

Background Управляет фоном элемента

Это универсальное CSS свойство позволяет одновременно задать несколько параметров фона, а именно определить цвет, установить фоновое изображение, задать начальное положение фонового изображения, определить как будет повторяться фоновое изображение, установить будет ли прокручиваться фоновое изображение вместе с содержимым и масштабировать фоновое изображение согласно заданным размерам. Так же значением свойства background может быть указана градиентная заливка фона.

Background-color - определяет цвет фона элемента

ПО УМОЛЧАНИЮ(ПРОЗРАЧНЫЙ)

.block{background: transparent;}

ЦВЕТНОЙ

.block{ background: red;}

ЦВЕТНОЙ(RGBA, ПРОЗРАЧНОСТЬ 50%)

.block{ background: rgba(86, 255, 80,0.644) }

Background-image - устанавливает фоновое изображение для элемента либо задает градиентную заливку

КАРТИНКА

.block
{background: url("../img/img/")}

РАДИАЛЬНЫЙ ГРАДИЕНТ

.block
{background: radial-gradient(circle, rgba)}

ЛИНЕЙНЫЙ ГРАДИЕНТ

.block
{background: linear-gradient()}

Background-repeat - определяет, как будет повторяться фоновое изображение

ПО УМОЛЧАНИЮ. ФОНОВОЕ ИЗОБРАЖЕНИЕ ПОВТОРЯЕТСЯ ПО ГОРИЗОНТАЛИ И ВЕРТИКАЛИ.

.block{background:url() repeat;}

NO-REPEAT. УСТАНАВЛИВАЕТ ОДНО ФОНОВОЕ ИЗОБРАЖЕНИЕ В ЭЛЕМЕНТЕ БЕЗ ЕГО ПОВТОРЕНИЙ

.block{background:url() no-repeat;}

REPEAT-X. ФОНОВЫЙ РИСУНОК ПОВТОРЯЕТСЯ ТОЛЬКО ПО ГОРИЗОНТАЛИ.

.block{background:url() repeat-x;}

REPEAT-Y. ФОНОВЫЙ РИСУНОК ПОВТОРЯЕТСЯ ТОЛЬКО ПО ВЕРТИКАЛИ.

.block{background:url() repeat-y;}

SPACE. ИЗОБРАЖЕНИЕ ПОВТОРЯЕТСЯ СТОЛЬКО РАЗ, ЧТОБЫ ПОЛНОСТЬЮ ЗАПОЛНИТЬ ОБЛАСТЬ; ЕСЛИ ЭТО НЕ УДАЁТСЯ, МЕЖДУ КАРТИНКАМИ ДОБАВЛЯЕТСЯ ПУСТОЕ ПРОСТРАНСТВО.

.block{background:url() space;}

ROUND. ИЗОБРАЖЕНИЕ ПОВТОРЯЕТСЯ ТАК, ЧТОБЫ В ОБЛАСТИ ПОМЕСТИЛОСЬ ЦЕЛОЕ ЧИСЛО РИСУНКОВ; ЕСЛИ ЭТО НЕ УДАЁТСЯ СДЕЛАТЬ, ТО ФОНОВЫЕ РИСУНКИ МАСШТАБИРУЮТСЯ.

.block{background:url() round;}

Background-position - задает начальное положение фонового изображения, где первое значение (проценты, px, слово) определяет расположение по горизонтали, второе - вертикали.

ПО УМОЛЧАНИЮ

.block{background: url() top left no-repeat;}

ПО ЦЕНТРУ

.block{background: url() center no-repeat;}

СЛЕВА

.block{background: url() left no-repeat;}

В УГЛУ

.block{background: url() left bottom no-repeat;}

ПОЗИЦИЯ

.block{background: url() 39px 25px no-repeat;}

Background-attachment - устанавливает, будет ли прокручиваться фоновое изображение вместе с содержимым элемента.

ПО УМОЛЧАНИЮ. ПОЗВОЛЯЕТ ПЕРЕМЕЩАТЬСЯ ФОНУ ВМЕСТЕ С ОКНОМ БРАУЗЕРА

.block{background: url() 0 0 no-repeat scroll;}

FIXED. ДЕЛАЕТ ФОНОВОЕ ИЗОБРАЖЕНИЕ ЭЛЕМЕНТА НЕПОДВИЖНЫМ, А ТАК ЖЕ УСТАНАВЛИВАЕТ ФОНОВОЕ ИЗОБРАЖЕНИЕ ОТНОСИТЕЛЬНО ОКНА БРАУЗЕРА

.block{background: url() center repeat fixed}

LOCAL. ФОН ФИКСИРУЕТСЯ С УЧЁТОМ ПОВЕДЕНИЯ ЭЛЕМЕНТА. ЕСЛИ ЭЛЕМЕНТ ИМЕЕТ ПРОКРУТКУ, ТО ФОН БУДЕТ ПРОКРУЧИВАТЬСЯ ВМЕСТЕ С СОДЕРЖИМЫМ, НО ФОН ВЫХОДЯЩИЙ ЗА РАМКИ ЭЛЕМЕНТА ОСТАЁТСЯ НА МЕСТЕ.

.block{background: url() center/cover no-repeat;}

Background-size - масштабирует фоновое изображение согласно заданным размерам. в цифрах, где первое значение (проценты, px) определяет ширину, второе - высоту. либо специальным значением.

ПО УМОЛЧАНИЮ. РАЗМЕРЫ КАРТИНКИ ПО УМОЛЧАНИЮ

.block{background: url() 0 0/ auto auto no-repeat;}

ПО ШИРИНЕ. РАЗМЕРЫ КАРТИНКИ ПО ШИРИНЕ БЛОКА

.block{background: url() 0 0 / 100% auto no-repeat;}

.block{background: url() 0 0 / auto 100% no-repeat;}

РАССТЯНУТЬ. РАССТЯГИВАЕМ КАРТИНКУ НА ВСЮ ШИРИНУ И ВЫСОТУ БЛОКА

.block{background:url() 0 0/ 100% 100% no-repeat; }

CONTAIN. МАСШТАБИРУЕТ ИЗОБРАЖЕНИЕ С СОХРАНЕНИЕМ ПРОПОРЦИЙ ТАКИМ ОБРАЗОМ, ЧТОБЫ КАРТИНКА ЦЕЛИКОМ ПОМЕСТИЛАСЬ ВНУТРЬ БЛОКА. ТО ЕСТЬ ПО МЕНЬШЕЙ СТОРОНЕ КАРТИНКИ

.block{background: url() 0 0 / contain no-repeat; }

COVER. МАСШТАБИРУЕТ ИЗОБРАЖЕНИЕ С СОХРАНЕНИЕМ ПРОПОРЦИЙ ТАК, ЧТОБЫ ЕГО ШИРИНА ИЛИ ВЫСОТА РАВНЯЛАСЬ ШИРИНЕ ИЛИ ВЫСОТЕ БЛОКА. ТО ЕСТЬ ПО БОЛЬШЕЙ СТОРОНЕ КАРТИНКИ

.block{background: url() 0 0 / cover no-repeat; }

РАЗМЕРЫ. УСТАНАВЛИВАЕМ КОНКРЕТНЫЕ РАЗМЕРЫ. ЕСЛИ УСТАНОВЛЕНО ОДНО ЗНАЧЕНИЕ, ОНО ЗАДАЕТ ШИРИНУ ФОНА, ВТОРОЕ ЗНАЧЕНИЕ ПРИНИМАЕТСЯ ЗА AUTO. ПРОПОРЦИИ КАРТИНКИ ПРИ ЭТОМ СОХРАНЯЮТСЯ. ИСПОЛЬЗОВАНИЕ ДВУХ ЗНАЧЕНИЙ ЧЕРЕЗ ПРОБЕЛ ЗАДАЕТ ШИРИНУ И ВЫСОТУ ФОНОВОЙ КАРТИНКИ

.block{ background: url() 0 0 / 40px 60px no-repeat;}
